# Vende tutti

Es un e-commerce de ventas en general



![preview](preview.gif)



## Instalación

 Para la instalación es necesario [node.js](https://nodejs.org/es/)



1. Clona el proyecto

   `git clone https://github.com/JonathanEmanuel/vende-tutti.git`

2. Instala dependencias

   `npm install`

3. Ejecuta el proyecto

   `npm start`



## Dependencias

- Bootstrap
- React Router Dom
